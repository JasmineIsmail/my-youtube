# 🎥 YouTube Clone (React.js)

A fully responsive, feature-rich YouTube clone built using React. This project replicates core YouTube frontend functionalities, including dynamic video streaming, real-time data fetching from the YouTube API, search auto-suggestions, and an optimized user interface.

---

## 🚀 Live Demo
🔗 https://vercel.com/jasmine10/my-youtube/4p5LisVSPb8m1cgSF7jneLLDSvBb

## ✨ Key Features
* **Dynamic Video Feed:** Fetches and displays real-time trending videos using the YouTube Data API.
* **Debounced Search & Suggestions:** Implements a high-performance search bar with debouncing to prevent API spamming and provide instant search suggestions.
* **Responsive Layout:** Tailored with Tailwind CSS for pixel-perfect display across mobile, tablet, and desktop screens.
* **Secure API Management:** Custom environment configurations (`.env`) to keep third-party API credentials isolated from source control.

---

## 🛠️ Tech Stack & Architecture

* **Core:** React.js (Functional Components, Hooks)
* **State Management:** Redux Toolkit 
* **Routing:** React Router DOM
* **Styling:** Tailwind CSS
* **Build Tool:** Vite
